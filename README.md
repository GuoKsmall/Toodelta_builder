## 使用教程
- 下载本仓库压缩包
- 打开面板
- 上传压缩包
- 在`插件文件`解压,选择gbk编码
- `reload`ToolDelta
- 将`2.json`和`get.json`移入相应文件夹
- 然后输入`?`查看菜单
